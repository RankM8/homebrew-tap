# RankM8 Homebrew Tap

Homebrew tap for RankM8 apps.

## WhisperM8

```bash
brew install --cask rankm8/tap/whisperm8
```

Hotkey dictation and Claude/Codex agent-chat manager for macOS.
Distributed self-signed; the cask clears the Gatekeeper quarantine on install.
